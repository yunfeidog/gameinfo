# 我的世界信息显示GameInfo模组

## 功能介绍

GameInfo模组可以在游戏中显示当前的游戏信息，包括：

- 游戏信息展示(FPS、天数、时间、坐标、生物群系、版本)
- 装备信息展示
- 方块信息展示
- 生物信息展示
- 熔炉信息展示
- 区块永久加载
